ClarkAI-Assistant

A voice-enabled personal AI assistant inspired by Iron Man’s Jarvis, built with LiveKit Agents framework and Google’s realtime generative AI model.

Tech Stack: Python 3.10+, LiveKit Agents, Google Cloud APIs (Generative Language, Gmail, Calendar), OAuth 2.0, Home Assistant API, Async Python

Key Features:

  • Conversational, sarcastic butler-style AI persona with real-time voice interaction.
  • Weather queries, web search (DuckDuckGo via LangChain), multi-language voice responses.
  • Advanced calendar management integrated with Google Calendar API.
  • Home automation control via Home Assistant REST API.
  • Email handling including reading, sending, replying with Gmail API and attachments.
  • Noise cancellation for clear voice communication.
  • Persistent contextual sessions and memory for natural multi-intent conversations.
  • Fully asynchronous design for efficient resource usage.

Setup & Usage: Virtual environment with dependencies installed via pip, environment variables configuration for secure API keys, run via python3 agent.py console.

Impact: Provides hands-free, intelligent automation and communication capabilities, enhancing productivity and smart home control with an engaging AI persona.
